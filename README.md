# Car-Rental
Car Rental MERN App

### Dependencies
  ### server
  `cors`
  `dotenv`
  `express`
  `mongoose`
  `nodemon`
  ### project
    `@redux-devtools/extension`
    `antd`
    `axios`
    `react`
    `react-bootstrap`
    `react-dom`
    `react-icons`
    `react-redux`
    `react-router-dom`
    `react-scripts`
    `redux`
    `redux-thunk`
    `web-vitals`
###
Create a `.env` file or change to your `MongoDb Url`

###
After installing dependencies run npm start on project side and nodemon server on client side
